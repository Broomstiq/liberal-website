Notes d'Utilisation

Philosophie du Design

  • Minimalisme Noir & Blanc : Le design doit être discret pour laisser les animations parler
  • Mouvement au Cœur : Chaque élément interactif a des transitions fluides
  • Performance Critique : Lazy loading et optimisations pour chargement rapide des GIFs
  • Mobile-First : Design pensé d'abord pour mobile, puis enrichi pour desktop

Priorités d'Implémentation

  • Must-Have MVP : Header sticky, mosaïque GIFs avec lazy loading, filtres, formulaire contact
  • Phase 2 : Fond animé custom, animations hover boutons réseaux, double système filtres

Accessibilité

  • Contraste noir/blanc : 21:1 ratio (WCAG AAA)
  • Touch targets minimum 48x48px sur mobile
  • Navigation au clavier complète
  • Alt text pour tous les GIFs/images
